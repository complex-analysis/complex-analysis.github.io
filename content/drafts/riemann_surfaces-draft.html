Riemann surfaces: Examples

The following are some examples of multiple-valued functions. In each case, the branch is identified with a diferent color.

  • $f(z) = z^{1/2}$
  • $f(z) = z^{1/3}$
  • $f(z) = \sqrt{1-z^2}$
  • $f(z) = \frac{1}{\sqrt{1-z^2}}$
  • $f(z) = \arctan(z)$
